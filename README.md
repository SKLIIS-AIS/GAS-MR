## 📦 Installation

This article has been accepted by IJCNN 2026, and the original author's project link is https://github.com/yuy63434-tech/GAS-MR

1. Create the conda environment:

```bash
conda env create -f environment.yml
```

2. Activate the environment:

```bash
conda activate gasmr
```

---

## 🚀 Running

Run the main script:

```bash
python main.py
```

Modify the configuration in `main.py` to test GAS under different settings.
